# Writing UI Mapping v1
## Binding Interpretation for index.html

**Status:** DRAFT (Pending Admin Approval)  
**Authority:** Subordinate to Writing Constitution v1

---

## 1. Voice Separation

- Alice prose renders as free-flow text (no bubble).
- System artifacts render as labeled cards.
- No avatar crossover.

## 2. Silence Rendering

- Breathing dot indicates engagement.
- Ellipsis indicates restraint.
- No fallback text.

## 3. Artifact Labeling

- Every artifact has a neutral label.
- No conversational phrasing in artifacts.

## 4. Errors

- Errors render as system cards.
- Alice may explain after, not instead.

## 5. Sources

- Sources are hidden by default.
- Revealed only on request.
